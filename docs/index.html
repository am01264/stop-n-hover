StopNHover(domTarget, oOptions)

Simple toggle function between 2 classes based on scroll-positioning. If the element is in the viewable area, it gets the class options.classStop (defaults to snh-stop) Otherwise, if it's even a pixel out of view, it gets the class options.classHover (defaults to snh-hover)

Useful for mostly CSS designs that need a hovering menu, that starts fixed to the top.

Theoretically compatible back to IE5, although I'm unsure about the JSON type object instanciation.
